Предварительно перед выполнением задания установите NodeJS
Его вы можете загрузить через сыылку: https://nodejs.org/ru/download/
P.S Лучше выбрать LTS версию

1 задание выполнять строго в консоли
следующие задания выполняются только через вызов через консоль
также не забудьте про аргументы вызова приложения
Пример: node app 4 
node <- является первым аргуметом. в нём записан путь к самому приложению NodeJs
app <- является вторым аргуметом. В неё записан путь уже к файлу app.js 
4 <- третий аргумент, он то вам и понадобится. Вместо '4' может быть любой символ
